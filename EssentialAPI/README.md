# EssentialAPI
EssentialAPI is an API that add's all the functions for SCP SL coding in an easier way!
repo: https://github.com/FlorentinaSL/EssentialAPI

# How do I put it in my project?
Download the Nuget package, or download the dll from the releases page and add it

# The API is compatible for Exiled and LabAPI?
Yes, you can work on Exiled or in LabAPI it's just make easier the coding!
BUT it's important that the file is in the LabAPI plugins folder and NOT in the
Exiled one.

# What Are the functions?
Here are all the functions:  
FEATURES:
- CustomLog system
- Server Managment
- Spawner of schematics (with Project MER)
- Timing    
PLAYER:
- CustomBroadcast
- CustomInventoryManagment
- Teleport
